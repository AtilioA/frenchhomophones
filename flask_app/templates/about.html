This is a small Flask web application I built during 2019-2020 summer vacation. My inspiration was this website, which showed me interesting English homophones from time to time; I used it as my homepage (on random) until I created my own, for French.

To gather the data, I crawled through Wiktionary's category of French homophones with BeautifulSoup and parsed words URLs entries with WiktionaryParser (with a bit of manipulation of my own). Thus, I had successfully extracted meaning from Wiktionary, I just had to create a database and a web application for it.

SQL solutions are limited to 10k rows on Heroku (Wiktionary has more than 50k homophones listed), so I went with MongoDB at mLab/Atlas, which offers 0.5GB storage for free and is easy enough to integrate with Heroku.

Flask extended with PyMongo was enough to carry out database connection and the web application's core. For CSS, I used Bootstrap. Still did not mess with JavaScript...

Feel free to contribute through the GitHub repository .
