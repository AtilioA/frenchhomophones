This is a little Flask web application I built during 2019-2020 summer vacation. My inspiration was this website, which showed me interesting English homophones from time to time; I still use it as my homepage (on random).

I crawled through Wiktionary's category of French homophones with BeautifulSoup and parsed words URLs entries with WiktionaryParser. Thus, I had successfully extracted meaning from Wiktionary, I just had to create a database and a web application for it.

SQL solutions are limited to 10k rows on Heroku (Wiktionary has almost 60k homophones listed), so I went with MongoDB at mLab/Atlas, which offers 0.5GB storage for free and is easy enough to integrate with Heroku.

CSS template/framework is bulma from jgthms, everything else to the favicon is my work. Feel free to contribute here.
