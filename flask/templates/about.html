This is a little Flask web application I built during 2019-2020 summer vacation.

My inspiration was this website, which showed me interesting English homophones from time to time; I used it as my homepage (on random) until I created my own, for French.

I simply crawled through Wiktionary's category of French homophones with BeautifulSoup and parsed URLs with WiktionaryParser. This way, I had successfully extracted meaning from Wiktionary, I just had to create a database and a web application for it.
